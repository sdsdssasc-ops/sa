# تعافي — موقع جلسات تأمل واسترخاء

موقع كامل (واجهة + سيرفر + قاعدة بيانات) لحجز جلسات التأمل والاسترخاء، باللغة العربية، جاهز للنشر المجاني على Render.

## المزايا

- صفحة رئيسية عربية كاملة مع قسم فيديو تعريفي (ملف أو يوتيوب)
- تسجيل حساب وتسجيل دخول آمن (bcrypt + JWT في كوكي httpOnly)
- حجز جلسات (نوع الجلسة + التاريخ + الوقت + ملاحظات)
- لوحة مستخدم: عرض الحجوزات وإلغاؤها
- لوحة أدمن: إحصائيات + إدارة كل الحجوزات (تأكيد/إتمام/إلغاء) + قائمة المستخدمين
- صفحة مختصين
- قاعدة بيانات PostgreSQL — الجداول تتنشأ تلقائيًا عند أول تشغيل
- نشر بضغطة واحدة عبر ملف `render.yaml` (Blueprint)

## التقنيات

React 19 + TypeScript + Vite + Tailwind CSS + shadcn/ui — Express + PostgreSQL + JWT

## هيكل المشروع

```
├── src/                    # الواجهة الأمامية (React)
│   ├── config.ts           # ← عدّل هنا: الاسم، الجلسات، الأسعار، المختصون، الفيديو
│   ├── pages/              # الصفحات (الرئيسية، الدخول، الحجوزات، الأدمن…)
│   ├── components/         # مكونات مشتركة (شريط التنقل، الفيديو…)
│   └── hooks/useAuth.tsx   # إدارة حالة تسجيل الدخول
├── server/                 # السيرفر الخلفي (Express)
│   ├── index.js            # نقطة التشغيل
│   ├── db.js               # الاتصال بقاعدة البيانات + إنشاء الجداول
│   ├── auth.js             # التسجيل والدخول والحماية
│   ├── bookings.js         # الحجوزات
│   └── admin.js            # مسارات الأدمن
├── public/videos/          # ← ضع الفيديو التعريفي هنا باسم intro.mp4
├── render.yaml             # نشر تلقائي على Render (موقع + قاعدة بيانات)
└── .env.example            # مثال لمتغيرات البيئة
```

## النشر على Render (مجانًا)

**الدليل الكامل المصوّر خطوة بخطوة موجود في ملف `دليل-النشر-خطوة-بخطوة.md` المرفق مع المشروع.** المختصر:

1. ارفع المشروع على GitHub
2. في Render: **New ← Blueprint** واختر المستودع
3. اضغط **Apply** — وبس، كل شي يتساوى تلقائيًا

## التشغيل المحلي (للتطوير)

```bash
# تحتاج PostgreSQL محلي + ملف .env (انسخه من .env.example)
npm install
npm run dev:server   # السيرفر على :3001
npm run dev          # الواجهة على :3000 (تتصل بالسيرفر تلقائيًا)
```

## حساب الأدمن الافتراضي

- البريد: ما تحطه في `ADMIN_EMAIL` (افتراضيًا `admin@taafy.local`)
- كلمة المرور: ما تحطه في `ADMIN_PASSWORD` (افتراضيًا `admin123456`)

**غيّر كلمة المرور فور النشر!** (من Render ← Environment ← عدّل `ADMIN_PASSWORD` ← احفظ)
